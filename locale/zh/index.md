Node.js® 是一个基于 [Chrome V8 JavaScript 引擎](https://developers.google.com/v8/) 的 JavaScript 运行时。
Node.js 使用事件驱动、非阻塞的 I/O 模式，使之变得轻量而高效。Node.js 的包生态系统， [npm](https://www.npmjs.com/)， 是世界上最大的开源库生态系统。
